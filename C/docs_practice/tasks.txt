Իրականացնել Singly Linked List տվյալների կառուցվածքը հետևյալ մեթոդներով՝
+append - Ավելացնել data էլեմենտը ցուցակի վերջից.
+prepend - Ավելացնել էլեմենտը ցուցակի սկզբից.
+insertAfter - Ավելացնել data էլեմենտը node հանգույցից հետո.
deleteNode - Հեռացնել data էլեմենտը.
+clear() - Հեռացնել ցուցակի բոլոր էլեմենտները.
+isEmpty() - Վերադարձնել 1, եթե ցուցակը դատարկ է և 0` հակառակ դեպքում.
+getSize() - Վերադարձնել էլեմենտների քանակը.
+printList() - Տպել ցուցակի էլեմենտները.
